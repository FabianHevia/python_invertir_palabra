## python_invertir_palabra
Repositorio que explora distintas formas de invertir una cadena de texto, utilizando desde métodos básicos hasta técnicas más avanzadas. El objetivo es comparar la eficiencia, legibilidad y estilo de cada solución.
---

## Contenido

- Inversión de cadenas usando *slicing* (`[::-1]`)  
- Inversión mediante bucles (`for` / `while`)  
- Inversión con funciones recursivas  
- Inversión utilizando estructuras de datos como *stacks* o *listas*  
- Comparación de tiempos de ejecución entre métodos

---

## Objetivo

El objetivo de este proyecto es mostrar cómo un mismo problema puede resolverse de diversas formas en Python, enfatizando la **claridad**, **eficiencia** y **estilo del código**.
